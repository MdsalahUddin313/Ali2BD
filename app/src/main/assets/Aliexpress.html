Aliexpress product importer

Extract different types of data from Aliexpress: product name, reference, prices, descriptions, images, features, reviews, categories with structure and more

Contact us



We developed online service for our client where the main function was Alixpress API integration. Client wanted to export Aliexpress products and after that to import them via XML,CSV feeds into his internal system. So we called this service “Aliexpress product importer”.

The next Fields can be extracted from Aliexpress

 

  • product name
  • reference
  • all prices
  • descriptions
  • all images
  • features
  • reviews
  • number of starts
  • number of reviews
  • variants (combinations or modifications) from Aliexpress product page
  • categories with structure from Aliexpress

Facts

 

Facts about our “Aliexpress product importer” solution

  • It is Online service for data export & import from Aliexpress.
  • Aliexpress data could be exported in the next formats: XML or csv, excel or JSON.

 

Implemented functions

 
1) Extraction items with the next products fields (sku, price, name, description, images, brand).
2) Extraction of item features (table with key name -> key value structure).
3) Extraction of item variants or combinations (for example: products options like colors, sizes).
